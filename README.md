# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Two-player game (Player X and Player O)
- Detects winner correctly
- Displays draw message when no moves are left
- Restart game functionality
- Simple and user-friendly UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Player X starts the game
2. Players take turns clicking on empty boxes
3. First player to align 3 marks (row, column, or diagonal) wins
4. If all boxes are filled and no winner → it's a draw
